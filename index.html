수동로스팅에 참고 할 레시피를 선택해보세요. 자동로스팅도 가능합니다.

Pilot Roast Mode

안정적인 자동 로스팅을 통해 완벽한 결과를 제공합니다

Simple Roast Mode

초보자도 쉽게 시작할 수 있도록 설계된 간단하고 직관적인 로스팅 모드입니다

Balance Roast Mode

제한된 옵션 안에서도 균형 잡힌 로스팅 경험을 선사합니다

Expert Roast Mode

정밀한 제어와 완전한 자유를 통해 전문가다운 로스팅 경험을 제공합니다

Preheat

예열 상황을 표시합니다. 100%가 되면 로스팅을 시작 할 수 있습니다.

Fan1

로스팅 시작 시 Fan1의 값을 설정합니다.

Heater

로스팅 시작 시 Heater의 값을 설정합니다.

Fan2

로스팅 시작 시 fan2의 값을 설정합니다. 2.5 ~ 5 사이를 추천합니다.

Input Amount

투입되는 생두의 용량을 선택합니다.

Recipe Name

레시피의 이름을 설정합니다.

Bean Name

생두의 이름을 설정합니다.

Start

로스팅을 시작합니다. 예열이 완료되면 버튼이 활성화됩니다.

Start

예열을 생략하고 로스팅을 시작합니다.
